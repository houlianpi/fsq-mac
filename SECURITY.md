# Security Policy

## Supported Versions

Security fixes are applied to the latest released version of `fsq-mac`.

## Reporting a Vulnerability

If you believe you have found a security vulnerability, do not open a public GitHub issue.

Please report it privately through GitHub Security Advisories or contact the maintainer directly if a private reporting channel is listed on the repository profile.

When reporting, include:

- A description of the issue
- Steps to reproduce
- Potential impact
- Any suggested mitigation

We will review reports as quickly as possible and coordinate disclosure once a fix is available.
